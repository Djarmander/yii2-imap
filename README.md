yii2 Imap
==========
This library is a fork of https://github.com/yiioverflow/yii2-imap

Installation by composer
------------
```composer
{
    "require": {
       "roopz/yii2-imap": "dev-master"
    }
}

Or

$ composer require roopz/yii2-imap "dev-master"
```
